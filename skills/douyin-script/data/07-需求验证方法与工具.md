# 需求验证方法与工具

> 来源：需求验证方法研究结果 + 选题词根分类研究结果

---

## 一、选题验证三维度

选题是否有真实需求，从三个维度判断：

1. **搜索热度**：有多少人在主动搜索这个问题？
2. **用户互动质量**：同类内容的评论区在问什么？
3. **内容竞争度**：这个选题的供给是否已经饱和？

## 三、评论验证信号

**小红书CES评分体系**（反映互动权重）：
- 点赞 1分
- 收藏 1分
- **评论 4分**（权重较高）
- **转发 4分**
- **关注 8分**

**可关注的需求信号**：
- 评论区反复出现同一个问题 → 强需求
- 高赞评论在问"怎么办""怎么做到的" → 强需求
- 评论只是"666""说得好" → 弱需求，只是内容好不是需求强

## 四、常青选题 vs 热点选题

| 判断维度 | 常青选题 | 热点选题 |
|----------|---------|---------|
| 时效性 | 永久有效，一年后看还成立 | 短期爆发，错过即失效 |
| 搜索趋势 | 重复搜索量稳定 | 搜索量短期暴涨后断崖下跌 |
| 触发源 | 人性层面的持续痛点 | 外部事件驱动 |
| 风险 | 低，易批量复制 | 高，需要快速跟进 |

**判断操作**：问自己"这个内容半年后还有人搜吗？"能 → 常青，不能 → 热点。

**配比建议**：70% 常青内容 + 30% 热点内容。

## 五、选题词根提取方法

### 关键词拆解法
1. 一级关键词：领域核心大词（如"创业""自媒体"）
2. 二级关键词：子模块（如"创业"→"获客""变现""定位"）
3. 三级关键词：具体话题（如"获客"→"怎么让用户主动私信你"）

### 判断"独立选题"vs"子素材"
- 能围绕它讲满1-3分钟口播 → 独立选题
- 只能作为某段话里的一个例子 → 子素材

### 选题颗粒度标准
- **太粗**："赚钱""创业" → 无法直接出内容
- **合适**："为什么自律不能帮你赚钱" → 有明确指向
- **太细**："卫生纸卖一个亿" → 只是素材，不是选题
