# 选题公式与筛选标准

> 来源：获客型白皮书 第四章 + 选题SOP三阶段 + DELIVERR框架

---

## 一、倒推式选题法（核心公式）

```
你的产品优势 → 变成行业标准 → 倒推出客户的终极焦虑 → 形成选题
```

### 四步操作

| 步骤 | 操作 | 例子（网盘拉新） |
|------|------|-----------------|
| 1. 盘点底牌 | 我的产品核心优势是什么？ | 自动化分发系统+独家诱饵资源库 |
| 2. 设定排他标准 | 怎么让别人觉得我的模式最好？ | 靠系统拉新 vs 靠体力拉新，系统才是未来 |
| 3. 捕捉客户焦虑 | 客户最怕什么？ | 怕天天发帖被封号、怕累死累活不赚钱 |
| 4. 生成选题 | 焦虑 → 问题 → 选题 | 《为什么你做网盘拉新一个月还赚不到1000块》 |

## 二、5种选题公式

### 公式1：反常识否定型
- 模板：为什么你做X的方法全是错的？
- 例：《为什么你天天发帖做拉新，反而越做越穷》

### 公式2：行业定义型
- 模板：202X年，什么样的X才能Y？
- 例：《2026年，什么样的副业项目才能真正赚到钱》

### 公式3：警告排他型
- 模板：劝你现在千万别做X
- 例：《劝你现在千万别在抖音教干货，除非你想吸引白嫖党》

### 公式4：阶层划分型
- 模板：做X的人分两种，你是哪一种？
- 例：《做网盘拉新的人分两种：搬砖的和操盘的》

### 公式5：趋势断言型
- 模板：X的未来不属于Y，属于Z
- 例：《副业的未来不属于勤奋的人，属于会用系统的人》

## 三、双重筛选标准

### 第一关：选题四问（获客型专用）

| 问题 | 判断标准 |
|------|---------|
| 排他性 | 这个选题只有我能讲吗？我的独特经历/产品是否不可替代？ |
| 反向筛选 | 这个选题会得罪劣质客户吗？白嫖党看了会不会划走？ |
| 精准吸引 | 这个选题能让优质客户产生"说的就是我"的共鸣吗？ |
| 转化路径 | 这个选题最终能指向我的产品吗？逻辑链条是否通顺？ |

### 第二关：DELIVERR（内容质量）

| 维度 | 判断问题 |
|------|---------|
| D 交付物 | 看完能拿走一个具体认知吗？ |
| E 证据 | 观众凭什么信你说的？有真实案例或数据吗？ |
| L 低门槛 | 认知上容易接受吗？不要求观众有专业背景？ |
| I 真实场景 | 观众会说"这就是我现在遇到的事"吗？ |
| V 变体 | 已经把"那我这种情况呢"的答案写进去了吗？ |
| E 经济价值 | 能用一句话说清楚"这对观众有什么好处"吗？ |
| R 可复用 | 同结构能再出3-5条吗？能形成系列吗？ |

## 四、选题标注规范

每个通过筛选的选题标注以下信息：

- **选题公式**：属于5种公式中的哪一种
- **矩阵角色**：破冰/造梦/树敌/论证/立人设
- **常青/热点**：长期有效还是时效性强
- **目标情绪等级**：L1轻度共鸣 / L2中度刺激 / L3高强度冲突

## 五、普通选题 vs 获客型选题

| 维度 | 普通选题 | 获客型选题 |
|------|---------|-----------|
| 出发点 | 用户关心什么我就讲什么 | 我手里有什么锤子，我就证明满世界都是钉子 |
| 逻辑方向 | 从需求到内容 | 从产品到焦虑到内容 |
| 结果 | 同质化内容，谁都能讲 | 排他性内容，只有你能讲 |
