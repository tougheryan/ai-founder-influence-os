# 反常识与认知颠覆公式

> 来源：选题方法论——反常识板块 + 获客型白皮书

---

## 一、核心概念："正确的非共识"

观众在看你之前，脑子里已经对事物形成了刻板印象。这些刻板印象常常是错误的，经不起推敲。

你的工作：**把真实情况搞清楚，击碎这些刻板印象，建立"正确的非共识"。**

表现形态：开头用一个反常识观点吸引住对方，看完整个内容之后，对方发现你是对的。

---

## 二、刻板印象的常见模式

刻板印象通常是**简化的因果关系**：

| 模式 | 示例 |
|------|------|
| 有X就能Y | 有流量就能赚钱 |
| 做X就会Y | 涨粉快就能变现 |
| X是Y的原因 | 成功是因为努力 |

核心特征：**单一因果，忽略条件。**

---

## 三、"正确的非共识"四部分结构

```
a. 识别刻板印象：流量大就赚钱
b. 提供反例：涨粉最快的博主不赚钱
c. 给出新框架：赚钱取决于流量质量和变现路径的匹配度
d. 展开内容：定义"流量质量"和"匹配度"两个关键词，完成内容
```

---

## 四、三个可直接套用的公式

### 公式1：基础版
```
很多人以为X，其实真正的问题是Y。
```
例：很多人以为自己赚不到钱是因为不够努力，其实真正的问题是没有系统。

### 公式2：现象版
```
为什么很多人明明出现了X，结果却走向了Y？因为他们忽略了Z。
```
例：为什么很多人明明每天都很忙，结果却一直赚不到钱？因为他忙的是执行，不是系统。

### 公式3：纠偏版
```
你以为X是原因，其实X只是结果，真正的原因是Y。
```
例：你以为穷的人才会忙，其实忙只是结果，真正的问题是没有杠杆和系统。

---

## 五、用在获客型内容中的位置

反常识公式最适合放在：
1. **开头钩子**：用公式1或公式3直接打破认知
2. **第二步"否定传统做法"**：用公式2指出用户忽略了什么
3. **第三步"建立新标准"**：用四部分结构完整展开
