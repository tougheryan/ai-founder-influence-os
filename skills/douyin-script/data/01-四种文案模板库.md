# 四种文案模板库

> 来源：获客型文案白皮书 第七章

---

## 模板A：反常识颠覆型

```
【开场】一个反常识断言（打破认知，制造危机感）
"你在X领域做的所有努力，可能都在往反方向走。"

【中间-踩压旧模式】具体描述旧模式的痛苦
"大多数人怎么做...结果是什么...为什么这注定失败..."

【中间-建立新标准】给出你的标准答案
"真正做得好的人，只做一件事：..."

【中间-赋予意义】解释为什么新标准有效
"因为底层逻辑不一样...这不是在X，这是在Y..."

【结尾】升维收尾，不留CTA
"永远记住，X的本质不是Y，而是Z。剩下的，你自己悟。"
```

适用角色：破冰、树敌
情绪曲线：L3（开场冲击）→ L1（理性解释）→ L2（新认知兴奋）→ L3（收尾金句）

## 模板B：阶层划分型

```
【开场】把人分成两类
"做X的人分两种，一种在搬砖，一种在操盘。"

【中间-描绘低阶人群】让用户不想成为这种人
"第一类人...他们每天...结果..."（描绘痛苦和徒劳）

【中间-描绘高阶人群】让用户渴望成为这种人
"第二类人...他们只做...结果..."（描绘轻松和高效）

【中间-揭示分水岭】两种人的本质区别
"区别不在于努力，而在于..."

【结尾】身份锚定，不留CTA
"你选择做哪一种人，决定了你的天花板在哪。"
```

适用角色：树敌、造梦
情绪曲线：L2（好奇分类）→ L1（低阶共鸣）→ L2（高阶渴望）→ L3（身份冲击）

## 模板C：案例暗度陈仓型

```
【开场】讲一个具体案例/场景
"你看我现在在这家XX，如果你要做XX..."

【中间-借案例分析行业问题】
"大多数人做XX的时候会犯一个错误..."

【中间-把你的产品包装成案例结论】
"真正应该做的是..."（描述的特征恰好是你产品的特征）

【结尾】升维到行业认知
"这就是我说的：X的本质不是Y，而是Z。"
```

适用角色：论证、立人设
情绪曲线：L1（场景代入）→ L2（问题揭示）→ L2（方案兴奋）→ L3（认知升维）

---

## 模板D：趋势断言型

```
【开场】对未来做出一个确定性的判断
"202X年，X领域会发生一个巨大的变化。"

【中间-论证为什么旧模式会死】
"现在的主流做法...但它有三个致命问题..."

【中间-描述新趋势的特征】
"新的模式一定是...具备X、Y、Z特征..."

【中间-给出具体操作方向】
"所以你现在就应该..."

【结尾】制造紧迫感但不给CTA
"等到所有人都看明白的时候，机会就已经没了。"
```

适用角色：破冰、造梦
情绪曲线：L3（趋势冲击）→ L1（逻辑论证）→ L2（希望感）→ L3（紧迫收尾）

## 模板选择决策表

| 选题类型 | 优先模板 |
|---------|---------|
| 反常识观点、认知颠覆 | 模板A |
| 人群划分、鄙视链建立 | 模板B |
| 个人案例、项目拆解 | 模板C |
| 行业趋势、未来判断 | 模板D |

## 两种写作姿态：论述流 vs 观察流

四种模板都可以用两种不同的写作姿态来执行：

### 论述流
直接陈述观点、论证、给出结论。适合趋势断言、干货论证、明确行动号召类内容。
- 例："你做X是错的，因为Y。正确做法是Z。"
- 用户感觉：被说服、被教导

### 观察流
通过高密度具体细节呈现现实，让用户自己推导结论。适合社会阶层、文化批判、反常识颠覆类内容。
- 例："你看挣的人走路是什么状态。再看赚的人怎么坐在咖啡馆里。这两个字之间隔着的不是钱，是阶级。"
- 用户感觉：被打开了一双新的眼睛

**选择判断**：
- 选题越接近社会观察/阶层分析/文化批判 → 越适合观察流
- 选题越接近趋势判断/操作指南/案例拆解 → 越适合论述流
- 模板A（反常识颠覆型）天然兼容两种姿态，是最灵活的结构

观察流详细技法见 `data/08-内容结构扩展库.md` 第六章。
