# 定位与人设公式

> 来源：自媒体小白从0到百万粉指导白皮书 第一、二章 + 获客型白皮书 第五章

---

## 一、定位公式

```
我是【某种经历/身份】的人，我持续为【某类人】讲清楚【某类问题】，并给出【某种结果】。
```

示例：
- 我是产品经理出身的商人，我持续为想做自媒体和创业的年轻人讲清楚内容增长和商业化问题，并给出可执行的路径。

---

## 二、定位必须回答4个问题

- 我是谁
- 我讲什么
- 我给谁讲
- 用户为什么要关注我（而不是别人）

---

## 三、定位不能太粗，也不能太细

- **太粗**："我分享成长""我讲商业" → 没人知道你具体解决什么
- **太细**："我专门讲南京某区23岁男生的某个极细问题" → 流量池太小
- **正确**：账号定位中等颗粒度 + 单条内容颗粒度更细

---

## 四、赛道选择四要素

| 要素 | 判断标准 |
|------|---------|
| 需求强度 | 用户会主动搜索？会在评论区反复问？愿意付费？ |
| 持续供给能力 | 你能不能连续讲100条不空？有没有真实经验和方法？ |
| 商业化空间 | 存在广告主？能带货？能做咨询/服务/产品？ |
| 长期性 | 半年后还有人搜？一年后还能继续讲？ |

## 五、获客型人设的三个战略问题

### 1. 你的"行业公敌"是谁？

- 必须是一个大家都看得见、很多人在踩坑的旧模式
- 公敌必须是主流做法（这样才有足够多人共鸣）
- 公敌必须有明显缺陷（这样攻击才站得住脚）
- 攻击公敌 = 间接证明你的独特性

例：经纬的公敌是"传统写字楼咖啡店"

### 2. 你的"排他性新标准"是什么？

格式：**X的本质不是Y，而是Z。**

- "网盘拉新的本质不是拉新，而是人性饵料与自动化流量系统的搭建"
- "自媒体变现的本质不是涨粉，而是筛选愿意付费的精准人群"

### 3. 你如何建立"高位框架"？

从"卖货的推销员"变成"制定规则的行业教父"，说话方式是关键：

| 低位框架 | 高位框架 |
|---------|---------|
| 求着客户买 | 告诉客户世界的规则是什么 |
| "我们的产品很好" | "这个行业的本质是这样的" |
| 让客户做裁判 | 你来做裁判，客户是学生 |

## 六、小白最适合切入的赛道类型

| 类型 | 适合人群 | 例子 |
|------|---------|------|
| 问题解决型 | 有明确经验和解决方案的人 | 自媒体起号、AI提效、求职转行 |
| 经历记录型 | 还没有方法论，但有真实过程的人 | 30天起号记录、从0做副业 |
| 认知输出型 | 有长期思考、能做深度解释的人 | 商业认知、创业心理、行业判断 |
| 服务承接型 | 已有服务或项目的人 | 咨询、陪跑、培训 |
