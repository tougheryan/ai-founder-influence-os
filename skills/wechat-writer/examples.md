# wechat-writer · 输入/输出示例

## 示例 1：从 breakdown 输出到公众号长文

### 输入（breakdown 摘要）

```markdown
## 关键词矩阵
- 核心词：AI 编程助手
- 情绪词：兴奋、焦虑、被取代
- 场景词：凌晨改 bug、实习生、老工程师

## 标题方案
1. 《Codex 来了，我为什么不慌》
2. 《我用 Codex 写了一个完整的后台》

## 开场钩子库
- "凌晨两点，我在网吧写了一晚上代码。"
- "坦白说，第一次看到 Codex 输出完整函数的时候，我愣住了。"
```

### 预期输出（公众号长文片段）

```markdown
---
step: write
slug: codex-night
date: 2026-06-23
source: breakdown:pipeline/breakdown/2026-06-23-codex-breakdown.md
---

# Codex 来了，我为什么不慌

凌晨两点，我在网吧写了一晚上代码。

不是因为我勤奋，是因为那个 bug 死活搞不定。同事都睡了，只有风扇在转。我盯着屏幕，突然想：要是能有个实习生帮我先把骨架搭出来，我只用改关键部分，该多好。

两个月后我见到了 Codex。

坦白说，第一次看到它输出完整函数的时候，我愣住了。不是因为它写得完美，而是因为它写得像一个真的实习生：有思路，有漏洞，有可以接着聊的余地。

...

以上，既然看到这里了，如果觉得不错，随手点个赞、在看、转发三连吧...
```
